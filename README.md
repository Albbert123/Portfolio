## 1. ecoFarma

**Final Degree Project — online pharmacy platform with semantic search capabilities**

**Overview**
ecoFarma is my final degree project: an online pharmacy platform designed to improve product discovery through semantic search. The core idea behind the project is to go beyond keyword-based search by using embeddings to capture the meaning of user queries and surface more relevant pharmaceutical products.

**What it does**

* Online pharmacy-style product browsing experience
* Semantic search powered by embeddings
* Improved discovery of products based on intent rather than exact keywords
* Full-stack architecture separating frontend and backend services

**Tech stack**

* **Frontend:** TypeScript, Next.js
* **Backend:** Python, FastAPI

**Repository:** https://github.com/Albbert123/ecoFarma
